## 功能特点

- 多个与中国社会和政治相关的问题
- 即时反馈和社会信用分数计算
- 响应式设计，适合各种设备
- 中文界面

## 如何使用

1. 打开 `index.html` 文件在浏览器中运行测试。
2. 回答所有问题，每个正确答案将获得 25 点社会信用分。
3. 回答错误将导致巨大的社会信用分扣除。
4. 测试结束后，您将看到最终的社会信用分数。
